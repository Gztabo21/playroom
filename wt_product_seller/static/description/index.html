WEBSITE PRODUCT SELLER

Key Features

  • Using this app you can create seller and seller can manually create sale order for other cutomer

Highlights

  • After installing the app, go to the settings of the root menu
  • Inside settings Go to User menu and click on the Create button.
  • Inside the user you have to select "Is seller" boolean field.
  • Inside website login with the user which you have created.
  • After login go to the Shop page. Then you can see the customer selection.
  • After select customer and product click on Add to Cart button.
  • Then see your selected customer name and address.
  • Click to confirm order on Confirm button.
  • You can see the sale order of the product which you have created with the customer.
