Promotion And Scheme Management

Video Tutorial

The business owners can now validate schemes and promotions for their product in a fully customizable manner with this module.

  • User can create reward scheme.

  • Scheme can be applied on product and product category.

  • User need to define product, minimum order qty and reward products and reward qty for the product.

  • User can define date range for scheme.

  • Scheme will be apply on quotation when user click on "Apply Scheme" at quotation level.

  • Sale order reward product will be manage on picking.

Prerequisites before Installation

No prerequisites required before installation.

  • Create promotion scheme for products.
  • User can define date range for promotion scheme.
  • User can create promotion scheme for product category too.
  • User can apply promotion on quotation by clicking on 'Update Scheme'.
  • User can view the rewarded product for particular product of whole sale order.
